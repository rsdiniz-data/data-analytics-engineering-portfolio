## 🚀 Projetos em Destaque

Os projetos abaixo representam a evolução de uma mesma solução de DRE, partindo de BI tradicional até uma arquitetura moderna de dados em nuvem.

---

### 📊 1. DRE Automatizada em Power BI  
🔗 https://github.com/rsdiniz-data/dre-analise-financeira-powerbi

Projeto de Business Intelligence focado na construção de uma DRE automatizada a partir de dados financeiros.

**Destaques:**
- Modelagem dimensional (Star Schema)  
- KPIs financeiros (Receita, EBITDA, Lucro)  
- Análises YoY, AH e AV  
- Simulações de cenário (What-If)  

---

### 🗄️ 2. Pipeline de Dados para DRE (SQL Server + Python)  
🔗 https://github.com/rsdiniz-data/dre-data-pipeline-sql-server

Evolução para uma arquitetura de dados estruturada, com ingestão, transformação e consumo centralizado.

**Destaques:**
- Ingestão de dados com Python (PDF e Excel)  
- Armazenamento em SQL Server  
- Modelagem em camadas (staging e analítica)  
- Transformações centralizadas no banco  
- Redução de processamento no Power BI  

---

### ☁️ 3. Data Lake Pipeline para DRE (PySpark + Arquitetura Medallion)  
🔗 https://github.com/rsdiniz-data/dre-data-lake-pipeline-pyspark

Evolução para uma arquitetura moderna em nuvem, com processamento distribuído e organização em Data Lake.

**Destaques:**
- Arquitetura Medallion (Bronze, Silver e Gold)  
- Processamento distribuído com PySpark  
- Pipeline automatizado e orquestrado  
- Armazenamento em formato columnar (Parquet)  
- Separação entre ingestão, transformação e consumo  
- Base preparada para escalabilidade e reuso
